# Database for a gym

A goal of this assignment is to develop a backend for a gym maintaining a database of:
- gym members
- gym classes
- member-class information


## How to run this application

```bash
# Install dependencies
$ npm install
$ npm install mongoose
$ npm install -g nodemon
$ npm install express
```

```bash
# Start backend app
$ npm start
```
